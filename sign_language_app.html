Sign Language Recognition

Sign Language Recognition merupakan aplikasi berbasis mobile yang memiliki fitur bahasa isyarat. User dapat mengetahui huruf apa saja yang mereka peragakan melalui gestur tangan sesuai bahasa isyarat di Indonesia. Terdapat banyak fitur lainnya yang bisa dipakai oleh pengguna aplikasi ini. User Experience serta User Interface nya pun sangat sederhana dan mudah untuk digunakan oleh pengguna.

Fitur:

  • Gestur Bahasa Isyarat
  • Speech to Text
  • Panduan Gestur Tangan Berupa Foto
  • Konversi Gestur Tangan Menjadi Teks

Teknologi:

  1. Python (Bahasa Pemograman)
  2. Machine Learning
  3. Numpy
  4. ML Kit
