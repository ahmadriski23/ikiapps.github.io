Aplikasi Suara Konsumen

ASK App merupakan aplikasi pengaduan masyarakat secara online. Masyarakat atau pengguna dapat mengadu barang yang mereka beli secara online. Dengan berbagai fitur yang sangat memadai membuat pengguna nyaman menggunakan Aplikasi Suara Konsumen ini. User Interface dan User Experience yang ditawarkan sangat menarik perhatian pengguna untuk digunakan dalam waktu yang cukup lama.

Fitur:

  • Login Menggunakan Email atau Google
  • CRUD Story
  • Like dan Comment Story
  • Profil Pengguna
  • Push Notification
  • Chat Sesama Pengguna

Teknologi:

  1. Flutter (Framework)
  2. Dart (Bahasa Pemograman)
  3. Provider (State Management)
  4. Firebase Cloud (Cloud Storage)
  5. Firebase Auth
